# Pages

Full page components organized by feature/user type:

- `admin/` - Admin dashboard and management pages (ManageUsers, General)
- `auth/` - Login and authentication pages (Login, rbac)
- `public/` - Public-facing pages (Home, About, Analytics)
- `survey/` - Survey creation and management pages (SurveyBuilder, CreateChallengeSurvey, AllSurveys)
- `user/` - User account and settings pages (Settings)
- `leader/` - School/district leader dashboards (placeholder for future use)

Each page should handle its own data fetching and state management.
